h1. "iNaturalist":http://github.com/inaturalist/inaturalist/tree/master powers "iNaturalist.org":http://inaturalist.org/

This application provides:
* A taxonomy system geared toward species
* A journal system
* Flickr linking
* Species explorer (uses the wikimedia api)
* ID help system
* Places support
* Observation endpoints for:
** Google Maps, Google Earth, CSV, and Atom
